1) Для взаимодействия с базой данных, необходимо создать ее с именем "practiksavichecv" затем импортировать в нее файл "practiksavichecv.sql".
2) Для входа в режим админа на сайте необходи нажать на кнопку"режим админа", затем ввести логин '123' и пароль '123'. 
3) Для добавлении книги на сайт необходимо нажать на кнопку "Добавить книгу", затем ввести все необходимые данные и нажать на кнопку добавить.
4) Для редактирование и удаление книги необходимо зайти в просмотр книги и нажать на кнопку "редактировать".
